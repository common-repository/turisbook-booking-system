Connect your website with Turisbook platform.

== Description ==
With Turisbook Booking System you can connect to Turisbook platform. You can show your rooms or apartments, search for availabilities and the client can book directly in your website.
A Turisbook account is needed to use this plugin.


== Installation ==
1. Install Turisbook Booking System either via the WordPress.org plugin directory, or by uploading the files to your server
2. Activate the plugin through the \'Plugins\' menu in WordPress
3. Go to Tools > Turisbook Booking System and login.
4. Set your Prebook and Confirmation page.
5. You can use various shortcodes in your pages
